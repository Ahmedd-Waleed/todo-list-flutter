# To-Do List App

A simple To-Do List (Goals App) built with **Flutter**.

## Features

- Add new goals
- Mark goals as complete (tap to strikethrough)
- Swipe left to delete goals
- Auto timestamp for each goal
- Progress tracker (X/Y goals completed)
